# Anshuman JHA – Portfolio & Productivity Hub

A simple, single-page personal portfolio combined with a lightweight productivity hub, built with plain HTML, CSS, and JavaScript.

## Features

### Portfolio Card

- **Profile section**
  - Name, title, and short bio
  - Email link (`mailto:`)
  - Avatar placeholder with initial
- **Social links**
  - GitHub
  - Instagram
  - LinkedIn
  - Google Keep
- **Skills**
  - Displayed as badges (e.g., HTML, CSS, JavaScript, Designer)
- **Featured Projects**
  - Project title
  - Short description
  - External link (currently pointing to Google as a placeholder)

### Productivity Hub

A tabbed interface for basic productivity tools:

- **Tabs**
  - Todos
  - Notes
  - Links (active by default)
  - Images

- **Links**
  - Add an optional title and a URL
  - Display saved links in a list

- **Todos**
  - Add simple text-based tasks
  - Display tasks in a list

- **Notes**
  - Add a note with a title and content
  - Display notes as a list or cards (handled via `script.js`)

- **Images**
  - Add an image with an optional caption and an image URL
  - Display images in a grid

### Theme Toggle

- A theme toggle button (`☀`) in the top area switches the theme (actual behavior implemented in `script.js` and `style.css`).
